## Usage
```
const {Breadcrumb} = uskin;

let items = [{
  name: 'Home',
  href: '#home'
}, {
  name: 'Store',
  href: '#store'
}, {
  name: 'Phones',
  href: '#phone'
}];

ReactDOM.render(<Breadcrumb items={items}/>, document);
```
